# TorchApi

TorchApi is api for training and predicting model with pytorch.

It is written in Python using asynchronous framework Sanic.

It can run multiple tasks at the same time.

## Requirements
* Python 3.8.5
* Sanic 22.3.0
* `pip install -r requirements.txt`
